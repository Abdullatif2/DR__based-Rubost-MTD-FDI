# Data Folder

This folder contains lightweight sample CSV files included only for GitHub-friendly packaging and quick testing.

The full training datasets used for the main experiments were excluded from this package because they exceed GitHub's standard file-size limit.

Included files:
- `dataset_case57_training_mini.csv`
- `dataset_case14.csv`
- `dataset_case_training_tryout.csv`

For full experiments, regenerate the datasets using `save_dataset.py` or host the full CSV files using Git LFS or an external storage link.
